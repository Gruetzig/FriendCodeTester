# FriendCodeTester
bruteforces through nintendo 3ds friend codes 

### Usage

```python3 friendcode.py <friendcode>```

Example:
```python3 friendcode.py 461423505```

* Note that it only accepts 1-12 characters, no '-'

Output:

amount of cycles
